Thermal study of a re-entry space launcher
An in-depth exploration of heat distribution on a re-entry rocket, modeled using the Finite Element Method and variational formulation. This project blends theoretical mathematics and numerical matrix computations to solve partial differential equations describing thermal behavior. Implemented with FreeFEM and Python.

I learned to code numerical matrix analysis in Python, where I implemented finite element discretizations and solved large sparse linear systems resulting from the variational formulation of the PDE. The variational formulation was done using FreeFEM, a specialized C-based software for solving PDEs with finite elements.
